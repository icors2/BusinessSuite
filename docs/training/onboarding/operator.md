## Screens you'll use (day one)

1. **Workforce → Time Clock** — clock in at shift start
2. **MES → Operator Console** — start/stop operations, enter quantities
3. **MES → Placard** — read work-order traveler at your station

## Top tasks

1. Clock in before touching the shop floor console.
2. Select your workstation operation (e.g. `WS-LASER`).
3. Start cycle when work begins; stop when done and enter qty completed.
4. Clock out at end of shift.
