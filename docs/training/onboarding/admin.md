## Screens you'll use (day one)

1. **Products / Customers / Vendors** — master data
2. **CPQ → Catalog settings** — rate cards and pricing (Admin-only)
3. **QMS → Checklist Builder** — inspection templates
4. **CMMS → Assets** — asset and PM configuration
5. **Analytics → Dashboard** — system-wide health

## Top tasks

1. Maintain master data (products, customers, vendors).
2. Configure CPQ catalog formulas and rate cards.
3. Set up QMS templates and CMMS PM rules.
4. Manage user roles (via seed/DB in demo; production IAM TBD).
5. Oversee UAT sign-off using the [master tracker](../../uat/uat-master-tracker.md).
