## Screens you'll use (day one)

1. **Returns → Queue** — RMA intake and triage
2. **Returns → Detail** — approve, receive, resolve
3. **Sales → Orders** — lookup original shipment (read)

## Top tasks

1. Customer calls → locate shipped order line.
2. Request RMA with qty and reason within return window.
3. Approve or reject based on policy.
4. Receive returned goods into `RET-01`.
5. Resolve with REFUND (credit memo) or other disposition.
