# Sales Rep — Day-One Quick Start

## Login & role

| Field | Value |
|-------|-------|
| Email | `manager@arcncode.local` |
| Password | `Manager123!` |
| Functional persona | **Sales Rep** → RBAC **Manager** (CPQ + Sales) |

> In production, assign Sales Rep users the Manager role scoped to CPQ/Sales modules.

## Screens you'll use (day one)

1. **CPQ → Quotes** — create and manage quotes
2. **CPQ → Catalog** — product pricing reference
3. **Sales → Orders** — converted orders and shipment status

## Top tasks

1. Create quote for customer; add product and fabricated lines.
2. Recalc and send quote to customer.
3. Accept quote when customer approves.
4. Convert to sales order; track allocation and shipment.
