# WMS Events

| Topic | When |
|-------|------|
| `wms.inventory.received` | Stock received into a bin |
| `wms.inventory.moved` | Put-away between bins or pick from bin |
| `wms.inventory.shipped` | Stock shipped out of a bin |
| `wms.inventory.adjusted` | Cycle-count or adjustment with reason code |

All topics are exported in `ALL_WMS_TOPICS`.
