# Returns Library

Return Merchandise Authorization (RMA) handling linked to sales orders, WMS receiving, QMS non-conformance, and finance credit memos.

## Features

- RMA request against shipped sales order lines with return window enforcement
- Approval/rejection workflow
- WMS receive into designated returns bin
- Optional auto-creation of QMS non-conformance for quality-related returns
- Refund resolution via finance credit memo (AR credit)
